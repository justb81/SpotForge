# variants/cars/assets

Grafiken für CarForge. Hier liegen Icon, Splash, Logo, Hintergrund und die
Seltenheits-Kartenrahmen (`frames/`), referenziert in `../app.definition.ts`.

Platzhalter – echte Assets folgen. Erwartete Dateien:

- `icon.png` – App-Icon (1024×1024)
- `splash.png` – Splash-Screen
- `logo.png` – In-App-Logo
- `background.png` – Hintergrund
- `frames/{common,uncommon,rare,epic,legendary}.png` – Kartenrahmen
